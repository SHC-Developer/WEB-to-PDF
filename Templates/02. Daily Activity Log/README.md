# 업무 일지 템플릿

## 파일 구분

| 파일 | 용도 |
|------|------|
| **daily-log.json** | 업무 일지 **데이터** 형식 (metadata, entries). 스크립트·다른 앱에서 사용. |
| **daily-log-pages.json** | 에디터/PDF 앱에서 **불러올 수 있는** Page[] 형식. 앱에서 **템플릿 → 파일에서 불러오기**로 선택하세요. |

에디터에서 업무 일지 레이아웃을 쓰려면 **daily-log-pages.json**을 불러오세요.

## 필드 설명

| 필드 | 설명 |
|------|------|
| **metadata** | 일지 전체 정보 (작성자, 부서, 기간) |
| **entries** | 날짜별 일지 항목 배열 |
| **date** | 날짜 (YYYY-MM-DD) |
| **dayOfWeek** | 요일 (월/화/수/목/금/토/일) |
| **tasks** | 시간대별 업무 목록 |
| **time** | 시간대 (예: 09:00-10:00) |
| **description** | 업무 내용 |
| **category** | 구분 (업무, 회의, 교육, 외부, 기타) |
| **summary** | 해당일 요약 |
| **notes** | 비고 |
| **hoursWorked** | 근무 시간 |

## 사용 예

- `daily-log.json`을 `2026-02-업무일지.json` 등으로 복사 후 작성
- `entries`에 날짜별로 객체를 추가
- `_categories`와 `_usage`는 참고용이며, 출력/리포트 시 제외 가능
